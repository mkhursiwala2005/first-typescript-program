# TypeScript Interactive Average Calculator

## Overview

### This program is a command-line TypeScript application that calculates the average of user-provided numbers.

#### It demonstrates:

    Static typing in TypeScript

    Compile-time type checking
    
    Runtime input validation
    
    Asynchronous user input handling
    
    Basic control flow (loops and conditionals)

#### What the Program Does

    Prompts the user to enter how many numbers they would like to provide.
    
    Validates the input to ensure it is a valid number.
    
    If the input is invalid (e.g., text instead of a number), the program displays an error message and prompts the user again.
    
    Prompts the user to enter each number one at a time.
    
    Each entry is validated.
    
    If an invalid value is entered, the user is prompted again until a valid number is provided.
    
    Once all valid numbers are collected, the program:
    
    Calculates the sum
    
    Computes the average
    
    Displays the result
